== THEME USAGE ==

= Theme Features =
* Custom Festival Wire post type for festival news and updates
* Location-based content organization
* Artist and venue taxonomy support
* Community integration with bbPress forums
* Newsletter post type for email campaigns
* Custom breadcrumb navigation
* Responsive design optimized for mobile
* SEO-friendly structure
* WooCommerce compatibility

= Theme Customization =
All available options can be used from Appearance->Customize

= Community Features =
* Forum post integration
* User activity feeds
* Community comment system
* Remote login functionality
